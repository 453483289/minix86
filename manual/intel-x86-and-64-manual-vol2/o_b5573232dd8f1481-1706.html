VPCMPW/VPCMPUW—Compare Packed Word Values Into Mask

INSTRUCTION SET REFERENCE, V-Z

5-320 Vol. 2C

Operation

CASE (COMPARISON PREDICATE) OF

0: OP  EQ; 
1: OP  LT; 
2: OP  LE; 
3: OP  FALSE;
4: OP  NEQ ;
5: OP  NLT; 
6: OP  NLE; 
7: OP  TRUE;

ESAC;

VPCMPW (EVEX encoded versions)
(KL, VL) = (8, 128), (16, 256), (32, 512)
FOR j  0 TO KL-1

i  j * 16
IF k2[j] OR *no writemask*

THEN 

ICMP  SRC1[i+15:i] OP SRC2[i+15:i];
IF CMP = TRUE

THEN DEST[j]  1;
ELSE DEST[j]  0; FI;

ELSE 

DEST[j] = 0 

; zeroing-masking only

FI;

ENDFOR
DEST[MAX_KL-1:KL]  0

VPCMPUW (EVEX encoded versions)
(KL, VL) = (8, 128), (16, 256), (32, 512)
FOR j  0 TO KL-1

i  j * 16
IF k2[j] OR *no writemask*

THEN 

CMP  SRC1[i+15:i] OP SRC2[i+15:i];
IF CMP = TRUE

THEN DEST[j]  1;
ELSE DEST[j]  0; FI;

ELSE 

DEST[j] = 0 

; zeroing-masking only

FI;

ENDFOR
DEST[MAX_KL-1:KL]  0
