PSLLDQ—Shift Double Quadword Left Logical

INSTRUCTION SET REFERENCE, M-U

4-432 Vol. 2B

VPSLLDQ (VEX.256 and EVEX.256 encoded version)
TEMP  COUNT
IF (TEMP > 15) THEN TEMP  16; FI
DEST[127:0]  SRC[127:0] << (TEMP * 8)
DEST[255:128]  SRC[255:128] << (TEMP * 8)
DEST[MAX_VL-1:256]  0
