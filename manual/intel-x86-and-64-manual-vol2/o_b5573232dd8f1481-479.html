Real Address Mode Exceptions

GP(0) 

If any part of the operand would lie outside of the effective address space from 0 to 0FFFFH.

#NM

If CR0.EM[bit 2] = 1.
If CR0.TS[bit 3] = 1.

#UD 

If CPUID.01H:ECX.SSE3[bit 0] = 0.
If the LOCK prefix is used.
