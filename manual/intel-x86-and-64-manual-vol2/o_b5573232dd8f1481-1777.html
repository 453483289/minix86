VPMOVSDB instruction (EVEX encoded versions) when dest is memory

(KL, VL) = (4, 128), (8, 256), (16, 512)
FOR j  0 TO KL-1

i  j * 8
m  j * 32
IF k1[j] OR *no writemask*

THEN DEST[i+7:i]  SaturateSignedDoubleWordToByte (SRC[m+31:m])
ELSE *DEST[i+7:i] remains unchanged*

; merging-masking

FI;

ENDFOR

ELSE
